MarkdownPreviewPro 1.1.0
========================

New features
------------
- Smart live refresh that preserves scroll position (local HTTP server)
- Relative image resolution from the markdown file directory
- KaTeX math ($...$, $$...$$)
- GFM task lists, footnotes, YAML frontmatter stripping
- Sticky TOC sidebar
- Export HTML and PDF (Chrome headless)
- Cross-platform browser detection (macOS / Windows / Linux)
- Configurable output_dir, server_port, browser, feature toggles
- Optional editor ↔ preview scroll sync

Settings: Preferences → Package Settings → MarkdownPreviewPro → Settings
